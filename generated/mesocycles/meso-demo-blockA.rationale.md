# Rationale — Block A (demo, pressing emphasis)

*Generic, publishable rationale for the demo rolling block (`meso-demo-blockA`). It explains
the **why** behind the generated plan in the same voice a `/replan` pass would use — no athlete
identifying data. The plan itself is emitted by `library/demo/build.ts`.*

## The block in one line

A non-peaking, ~6-week **pressing-emphasis** block: strict-skill first while fresh, upper-body
hypertrophy biased to chest/delts/back, **polarized** conditioning (sprinty hits + a Zone-2 base),
legs and conditioning **maintained**, daily mobility + a wellness/breathing close-out — undulating
hard/easy with no back-to-back MetCons.

## Standing hip-health emphasis (snapping-hip management)

This block folds in an ongoing **hip-stability/mobility** thread — treated as standing programming
(like the unilateral / anti-rotation rail), not a phase. It targets benign **snapping-hip**
patterns (a tendon/soft-tissue snap that is usually painless), drawing on the best-evidenced
conservative care: **eccentric-biased strengthening** plus **targeted mobility** for deep-glute and
hip-flexor tightness. It is **stability/health work, not a lower-body hypertrophy bump** — legs stay
at their maintenance dose.

How it shows up in the generated sessions:

- **Daily warm-up opener** — glute-med activation (lateral band walk) + a light, controlled
  **Cossack squat** (quality ROM, never forced depth). The Cossack is the movement that most
  reliably helps, so it's a daily touch.
- **Dedicated hip day (mobility slot)** — a deeper **Cossack progression** alongside deep-glute
  (figure-4), iliopsoas (half-kneeling hip-flexor), and adductor release — the soft-tissue tightness
  conservative care prioritizes.
- **Eccentric strengthening, ~2×/week on easy days** — alternating the **external lever**
  (eccentric glute-med abduction — strongest evidence for the lateral/abduction snap) and the
  **internal lever** (eccentric iliopsoas lowering — the anterior/groin snap). Placed on easy days so
  it never competes with a hard skill/MetCon day; the iliopsoas is a tendon, so it ramps gently
  (≤~10%/week), consistent with "tendons are the governor."
- **An optional movement-screen note** (same spirit as the end-range-spinal-flexion caution): keep
  doing the conservative work, but get a clinician/screen if **red-flag** symptoms appear (painful
  locking, catching, giving-way, sharp deep-groin pain, a mechanical block, or progressive pain),
  which would point to an intra-articular cause rather than benign snapping. The note **does not gate
  training** — conservative work proceeds meanwhile.

## Calibration note

Skill/strength gate thresholds and starting loads remain placeholders until the Week 1–2 testing
block (see `docs/testing-protocol.md`); the hip work is dosed by **feel and quality ROM**, not by a
prescribed depth or load target — deliberately so, since chasing range is what aggravates a snapping
hip. Progress conservatively and let it be guided by what stays painless.
