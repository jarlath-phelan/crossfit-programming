# Open Programming

> An open-source, forkable system for **adaptive CrossFit + wellness programming** — one human-authored
> source of truth that compiles to both a printable program and an offline-first app, with a Claude Code
> agent that continuously re-plans against your logged history and daily readiness.

It is **CrossFit for wellness, longevity, and confidence** — not competition. The defaults bias toward
feeling good, moving well, getting strong, and looking good, rather than chasing PRs or peaking for a date.
